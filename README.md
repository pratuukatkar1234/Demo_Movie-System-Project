# Demo_Movie-System-Project

The Online Movie Ticket Booking System is a web-based application that allows users to browse movies, select show timings, and book tickets online. The system provides a simple and user-friendly interface for seamless ticket booking and management.

Features:

1) Movie Browsing
Users can view available movies along with details such as name, genre, and show timings.

2) Showtime Selection
Users can select their preferred show timings.

3) Seat Booking System
Users can choose and book seats based on availability.

4) User Registration & Login
New users can register and securely log in to the system.

5) Booking Management
Users can view their booking history and manage tickets.

6) Database Integration
Uses MySQL database to store user, movie, and booking data.

7) CRUD Operations
Supports Create, Read, Update, and Delete functionalities.

8) Form Validation
Ensures proper input validation on both client-side and server-side.

9) Responsive UI
Designed using HTML, CSS, and JavaScript for a user-friendly experience.

10) Backend Processing
Uses PHP for handling server-side logic.

Language used :
Frontend languages - 
1.	HTML 
2.	CSS 
3.	Bootstrap 
Backend languages – 
1.	PHP 
2.	JavaScript 
3.	jQuery 
4.	SQL
